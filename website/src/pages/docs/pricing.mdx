# Pricing

Holocron is **free to start**. Create a site, write your docs, and ship your **first production deployment** without paying anything. When you are ready to deploy continuously, **Holocron Pro** unlocks unlimited deploys, preview deployments, and the hosted AI assistant.

<Aside>
<Info>
Subscriptions are **per site**. Each project you deploy has its own plan, so you only pay for the docs sites you actively maintain.
</Info>
</Aside>

## Plans

| | **Free** | **Pro** |
| --- | --- | --- |
| Production deployments | 1 | Unlimited |
| Preview deployments | — | Every branch and PR |
| Hosted AI chat | Temporary model | Full model, no nag |
| Analytics | — | Coming soon |
| Price | $0 | **$99 / month** |

The **yearly plan** is **$990 / year** — two months free compared to paying monthly.

## What Pro unlocks

### Unlimited production deployments

On the **Free** plan you get a single production deployment so you can see your site live. **Pro** removes the limit, so every push to your default branch publishes a new version.

### Preview deployments

Every branch and pull request gets its **own preview URL** on Pro. Review docs changes in context before they reach production, the same way you review code.

<Aside>
<Tip>
Preview deployments are created automatically when **GitHub Actions** deploys from a PR branch. No extra configuration needed.
</Tip>
</Aside>

### Hosted AI chat

The built-in **AI assistant** answers questions about your docs using your content. Free sites use a temporary model with a small upgrade notice; **Pro** sites use the full model with no notice.

## Subscribe

Manage billing from the **Billing** tab of your project in the dashboard. Pick monthly or yearly, and switch or cancel any time through the Stripe billing portal.

<Aside>
<Note>
Billing runs on **Stripe**. You can update your payment method, download invoices, and cancel from the billing portal at any point.
</Note>
</Aside>

[Open the dashboard](/dashboard) to subscribe.
