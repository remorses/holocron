---
title: Getting Started
description: Build and deploy the minimal Cloudflare example.
---

## Install dependencies

```bash
pnpm install
```

## Run locally

```bash
pnpm --filter example-cloudflare dev
```

## Build the Worker bundle

```bash
pnpm --filter example-cloudflare build
```

## Deploy

```bash
pnpm --filter example-cloudflare deploy
```

<Note>
`wrangler deploy` reads the Cloudflare-ready output generated by Vite.
</Note>
