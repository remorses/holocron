## Overview

This example keeps the docs source in `pages/` instead of the project root.

It uses the regular `holocron()` plugin plus `@cloudflare/vite-plugin` so the same build can be deployed with `wrangler deploy`.

<Tip>
Use this folder as the smallest starting point for a Cloudflare-hosted Holocron site.
</Tip>
