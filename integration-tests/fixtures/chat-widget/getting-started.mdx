# Getting Started

Follow these steps to get up and running with Chat Test Docs.

## Installation

Install the package using npm:

```bash
npm install chat-test-docs
```

## Configuration

Create a configuration file in your project root:

```ts
export default {
  name: 'My Docs',
  theme: 'default',
}
```
