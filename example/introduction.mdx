## What is Holocron?

Holocron turns your MDX files into a beautiful documentation site. Point your `vite.config.ts` at the plugin and start writing docs.

## Features

- **MDX support** — write documentation in Markdown with embedded JSX components
- **Mintlify compatible** — drop-in replacement for `docs.json` configuration
- **Vite powered** — fast dev server with hot module replacement
- **React Server Components** — zero-JS pages by default, interactive where you need it

## Quick example

```ts
// vite.config.ts
import { defineConfig } from 'vite'
import { holocron } from '@holocron.so/vite'

export default defineConfig({
  plugins: [holocron()],
})
```

## Native details

Holocron also supports copied HTML `<details>` blocks and renders them with the same expandable UI.

<details open>
<summary>Show a **rich** summary with a [link](https://holocron.so)</summary>

Details can contain normal markdown content, including tables and code blocks.

| Component | Support |
| --------- | ------- |
| Tables    | Yes     |
| Code      | Yes     |

```ts
import { holocron } from '@holocron.so/vite'

export default {
  plugins: [holocron()],

```
